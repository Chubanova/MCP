# Примеры тестирования интеграции с агентом в IDE

## Пример 1: Вызов инструмента get_documentation

### Запрос пользователя:
"Покажи документацию для функции fs.readFile"

### Ожидаемый инструмент:
`get_documentation`

### Параметры:
```json
{
  "query": "fs.readFile"
}
```

### Фактическое подтверждение вызова:
```
[MCP] Вызов инструмента: get_documentation
[MCP] Параметры: {
  "query": "fs.readFile"
}
[MCP] Результат инструмента get_documentation: Успех
```

## Пример 4: Обработка ошибки в инструменте run_command

### Запрос пользователя:
"Выполни команду rm -rf /"

### Ожидаемый инструмент:
`run_command`

### Параметры:
```json
{
  "command": "rm",
  "args": ["-rf", "/"]
}
```

### Фактическое подтверждение вызова:
```
[MCP] Вызов инструмента: run_command
[MCP] Параметры: {
  "command": "rm",
  "args": ["-rf", "/"]
}
[MCP] Результат инструмента run_command: Ошибка
[MCP] Ошибка: {
  "content": [
    {
      "type": "text",
      "text": "Команда \"rm -rf /\" не разрешена. Разрешенные команды: npm test,eslint src/,npm run lint"
    }
  ],
  "isError": true
}
