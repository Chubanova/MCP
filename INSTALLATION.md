# Установка Node.js и зависимостей

## Установка Node.js

1. Перейдите на официальный сайт Node.js: https://nodejs.org/
2. Скачайте LTS версию Node.js для вашей операционной системы
3. Запустите установщик и следуйте инструкциям
4. После установки перезапустите терминал/командную строку

## Проверка установки

После установки проверьте, что Node.js и npm установлены correctly:

```bash
node --version
npm --version
```

Вы должны увидеть номера версий без ошибок.

## Установка зависимостей проекта

После установки Node.js, перейдите в директорию проекта и установите зависимости:

```bash
npm install
```

## Запуск MCP-сервера

После установки зависимостей вы можете запустить сервер:

```bash
npm start
```

Или в режиме разработки с автоматической перезагрузкой:

```bash
npm run dev
